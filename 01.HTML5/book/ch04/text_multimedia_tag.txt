하이퍼링크
    <a href="URL>   anchor의 약자. a 태그는 기본적으로 href 속성을 가지며, 이 속성값에 URL을 입력해 원하는 곳으로 이동한다.

    URL에 표시할 수 있는 값으로 다음의 다섯가지 유형이 있다.
        절대경로 URL 표기: href="http://www.w3.org/index.html"
        상대경로 URL 표기: href="hello.html"  (<base> 태그에 기본 URL 지정) --> 이게 뭔말이지?
        페이지 내 요소 표기: href="#top"
        프로토콜 URL 표기: https://, ftp:// ,mailto:, file: 등
        스크립트 표기: href="javascript:alert('Hello');"

    <a> 태그의 href 속성이 연결하고자 하는 웹 사이트의 URL을 지정하는 역할을 한다면,
    target 속성은 링크를 클릭했을 때 웹 사이트가 열릴 곳을 지정하는 역할을 한다.
    다음은 target 속성에 사용할 수 있는 속성값을 정리한 것이다.
        _blank : 새로운 웹 브라우저 창에 연다.
        _self : 현재 웹 브라우저 창에 연다(기본).
        _parent : 부모 웹 브라우저 창에 연다. (이게 뭐지?)
        _top : 웹 브라우저 전체 영역에 연다. (이건 또 뭐지?)

책갈피
    웹 문서의 내용이 길 때 같은 문서 내에서 원하는 위치로 바로 갈 수 있도록 해주는 기능을 책갈피라고 한다.
    책갈피는 <a> 요소의 name 속성 혹은 id(identifier) 속성을 이용해 문서 내 이동할 곳의 위치를 지정하고,
    <a> 요소의 href 속성값으로 '#name' 혹은 '#id' 값을 입력해 해당 위치로 링크를 건다.
    두 속성은 약간의 차이점이 있다. id 속성값을 쓸 때는 기본적으로 중복된 값이 없도록 주의해야 한다.
    하지만 name 속성은 중복이 허용되기 때문에 그룹으로 속성을 지정할 수 있다.

    이동 링크 지정 : <a href="#idname">idname 위치로 이동합니다. </a>
    이동할 곳의 위치 지정 : <p id="idname">이곳은 idname 위치입니다.</p>
    이동할 곳의 위치 지정 : <a href="idname">이곳은 idname 위치입니다. </a>
