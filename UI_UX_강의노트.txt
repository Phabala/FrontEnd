# 디자인

제가 직접 정리한 PPT, 홈페이지, 앱 등 공통적으로 디자인할 때 필요한 점과 사이트를 정리한 내용입니다, 프로젝트 시 활용하시면 매우 좋습니다, 실무에서 활용하시는 분들도 계십니다!

<1. 글씨체/폰트>
무료폰트 다운로드 사이트

- 한글(눈누) [noonnu.cc](http://noonnu.cc/)
- 영어(다폰트) [www.dafont.com](http://www.dafont.com/)

프로젝트 시 저작권은 생각하지 않아도됨, 다만 공모전, 블로그/SNS 업로드 등이 이루어질 경우 꼭 저작관 확인하셔야합니다!

눈누사이트는 모든폰트 메뉴클릭해서 사용하시고 필터에서 허용범위 모두 체크하고 사용하면 저작권 안심하고 쓰실 수 있고

다폰트는 다운로드 버튼 위에 100% Free만 상업적으로 사용 가능합니다

(좀 내려야 있어요)

[https://webgradients.com/](https://webgradients.com/)
그라데이션 색상 추천

색상을 쓰실때 진한색(어두운색이아님)을 바탕색으로 쓰시고 그 위에 흰색이나 밝은색 글씨나 아이콘 등 디자인요소를 넣어야 예쁘게 나옵니다

색상선택하실 때 참고해주세요!

<3. 아이콘/이미지소스>
[https://thenounproject.com/](https://thenounproject.com/)
컨셉맞추기 쉬운 무료 아이콘 다운로드

[https://www.flaticon.com/](https://www.flaticon.com/)
다양한 컨셉의 무료 아이콘 다운로드

[https://www.freepik.com/](https://www.freepik.com/)
무료 디자인소스 다운로드 (상업적이용불가)
*일러스트/포토샵 기본정도 아는사람 사용추천

아이콘을 사용하실 때 한페이지 안에 비슷한 컨셉으로 아이콘을 여러개 넣으실땐 위에있는 사이트가 편리하고 한두개씩 좀 예쁘게 넣기에는 아래 사이트가 좋을꺼에요

위 두개 사이트 아이콘 무료로 사용할때는 출처를 작게 사이트하단에 표시하면 좋습니다

<4. 사진/동영상>
[https://pixabay.com/](https://pixabay.com/)
사진데이터가 가장많은 무료사진 사이트

[https://unsplash.com/](https://unsplash.com/)
다운로드가 가장편한 무료사진 사이트
(원하는 사진 마우스오버 후 다운로드 버튼 클릭 시 바로다운로드)

[https://pexels.com/](https://pexels.com/)
무료 동영상 다운로드 사이트

사진은 가장 강력한 디자인 무기입니다
사용하실때 가로 크기가 1920px 이상 사진을 사용하셔야해요 보통 PC 모니터해상도의 가로가 1920px이기 때문입니다 (해상도가 낮은 사진(1920px)보다 낮은사진을 사용하면 흐리거나 깨져보입니다)
